Projet Personnel · Logiciel · Embarqué

Home Assistant sur Dell Wyse

Concevoir Implanter Linux Docker

Déploiement d'un véritable projet d'infrastructure embarquée low-cost. L'objectif était de concevoir un serveur domotique robuste pour centraliser la gestion du logement avec une consommation électrique minimale.

Architecture Matérielle

Le serveur est construit sur la base d'un "thin client" récupéré et modifié :

  • Base : Dell Wyse Dx0D.
  • Upgrades : Passage à 8 Go de RAM DDR3L et remplacement du stockage par un SSD SATA.
  • Refroidissement : Ajout de ventilateurs USB 40mm pour assurer un refroidissement actif 24/7 du système.

Infrastructure Logicielle

Le système repose sur un environnement Linux fiabilisé :

  • OS : Debian server (headless).
  • Conteneurisation : Utilisation de Docker pour isoler les services (Home Assistant, bases de données, broker MQTT éventuel).

Intégrations configurées

Développement annexe en cours

Conception et impression 3D d'une "enclosure" (boîtier personnalisé) en PETG/ASA. L'impression est réalisée sur une imprimante Anet A8 modifiée (firmware Marlin et contrôle réseau via OctoPrint).

Bilan des apprentissages

Ce projet concret est une excellente introduction à l'administration de systèmes Linux embarqués, à la gestion de conteneurs avec Docker, à la configuration de réseaux locaux complexes et bien sûr, à la logique d'automatisation domestique de bout en bout.
